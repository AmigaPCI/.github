The AmigaPCI is an OCS/ECS Amiga computer in the ATX form factor. It provides a flexible CPU Local Bus port, five PCI slots for expansion purposes such as video and sound, HID input, a dual port ATA controller, and other improvements over legacy Amiga designs.  

**Here you will find everything needed to build and use the AmigaPCI:**  
* **Links to all relevant project repos**  
* **Gerber files**  
* **RTL files for FPGAs and STM Microcontroller**  
* **Hardware documentation**  

If you are building the AmigaPCI from scratch, you should start at the [Building the AmigaPCI](/) document. If you already have a working board, you should review the [AmigaPCI Hardware Reference](https://github.com/AmigaPCI/AmigaPCI-MainBoard/blob/main/AmigaPCI_Hardware_Reference_Guide.md).
